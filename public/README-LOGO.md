# Logo E-Tawjihi

Pour ajouter le logo E-Tawjihi à la page de connexion :

1. Placez votre fichier logo dans ce dossier (`public/`)
2. Nommez-le `logo-e-tawjihi.png` (ou `.jpg`, `.svg`, etc.)
3. Le logo sera automatiquement affiché sur la page de connexion

**Formats supportés :** PNG, JPG, SVG
**Taille recommandée :** 200x80 pixels ou proportionnel

Si le logo n'est pas trouvé, un placeholder avec le texte "E-Tawjihi" sera affiché automatiquement.
